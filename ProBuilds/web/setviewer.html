How do I use it?

You can either download all the item sets or click on an item set on the left (ordered alphabetically by champion then lane). If you click on an item set it will then be displayed here and on the right you can copy and paste it to a file or download the item set file.

After you have an item set file or several you must navigate to your League of Legends installation directory. By default that is C:\Riot Games\League of Legends

You will then need to place the item set json file in the sub directory path Config\Champions\{championKey}\Recommended\ where {championKey} is the first name in the path on the right download pane. So for Aatrox it would be C:\Riot Games\League of Legends\Config\Champions\Aatrox\Recommended\.

If you downloaded all item sets you will want to extract the zip file to the sub directory path Config\Champions\ and then each folder, sub-folder and json file will be copied to the correct place.

There is a chance that the {championKey} or sub directory Recommended doesn't exist, if this is the case simply create it.

About Pro Item Sets

Pro Item Sets is made up of item sets generated by pulling top tier player match information from the Riot Games API and figuring out at different points in their game what was most commonly purchased.

It was developed by Kyle Schouviller and Nathan Bowhay for the Riot Games API Challenge 2.0 from August 10th to August 31st 2015.

For more information about the tool used to generate the item sets, check out the project on Github.
